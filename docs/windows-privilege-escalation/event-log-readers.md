## Security Logs

```pwsh
PS C:\Users\logger> wevtutil query-events Security /rd:true /f:text | Select-String /user
```

## Security Logs (Different User)

```pwsh
PS C:\Users\logger> wevtutil query-events Security /rd:true /f:text /r:SHARE01 /u:julie.clay /p:Welcome1 | Select-String /user
```
