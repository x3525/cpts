Dizide kaç bayt karakter olduğunu hesaplayalım:

```sh
my@attack:~$ echo -E "$CHARACTERS" | sed 's/\\x/ /g' | wc -w
```

```output title="Output"

```

Program breakpoint üzerinde durduktan sonra yığını görüntülemek için x komutunu (examine) kullanacağız.

Bu komut aşağıda verilen formatı takip eder:

```text title="Format"
x/nfu addr
```

| OPTIONAL PARAMETER | DESCRIPTION |
|---|---|
| n | Görüntülenecek bellek miktarı. |
| f | Görüntüleme formatı. Örneğin hexadecimal için x kullanılabilir. |
| u | Birim boyutu. Örneğin bayt için b kullanılabilir. |
| addr | Başlangıç bellek adresi. |
