# Tooling

## [Ffuf](https://github.com/ffuf/ffuf)

```sh
my@attack:~$ go install github.com/ffuf/ffuf/v2@latest
```

## [Gobuster](https://github.com/OJ/gobuster)

```sh
my@attack:~$ go install github.com/OJ/gobuster/v3@latest
```

## [Wenum](https://github.com/WebFuzzForge/wenum)

```sh
my@attack:~$ pyenv install 3.12
my@attack:~$ pyenv shell 3.12
my@attack:~$ pipx install --python 3.12 git+https://github.com/WebFuzzForge/wenum
my@attack:~$ pipx runpip wenum install setuptools
```

## [API Fuzzer](https://github.com/PandaSt0rm/webfuzz_api)

```sh
my@attack:~$ git clone https://github.com/PandaSt0rm/webfuzz_api.git
my@attack:~$ cd webfuzz_api
my@attack:~/webfuzz_api$ pip3 install -r requirements.txt
```
