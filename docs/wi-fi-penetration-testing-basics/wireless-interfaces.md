## Getting TX Power

```sh
my@attack:~$ iwconfig
```

```output title="Output" hl_lines="4"
lo        no wireless extensions.

wlp0s20f0u3  IEEE 802.11  ESSID:off/any
          Mode:Managed  Access Point: Not-Associated   Tx-Power=20 dBm
          Retry short limit:7   RTS thr:off   Fragment thr:off
          Power Management:off
```

## Changing TX Power

!!! danger

    Kablosuz arayüze ait varsayılan TX gücünü değiştirmek bazı ülkelerde yasa dışı bir işlem olarak kabul edilir.

```sh
my@attack:~$ sudo ifconfig wlp0s20f0u3 down
my@attack:~$ sudo iwconfig wlp0s20f0u3 txpower 30
my@attack:~$ sudo ifconfig wlp0s20f0u3 up
```

## Setting the Channel

```sh
my@attack:~$ sudo ifconfig wlp0s20f0u3 down
my@attack:~$ sudo iwconfig wlp0s20f0u3 channel 1
my@attack:~$ sudo ifconfig wlp0s20f0u3 up
```

## Setting the Channel (Automatic)

```sh
my@attack:~$ sudo ifconfig wlp0s20f0u3 down
my@attack:~$ sudo iwconfig wlp0s20f0u3 channel auto
my@attack:~$ sudo ifconfig wlp0s20f0u3 up
```

## Setting the Frequency

```sh
my@attack:~$ sudo ifconfig wlp0s20f0u3 down
my@attack:~$ sudo iwconfig wlp0s20f0u3 freq "2.412"
my@attack:~$ sudo ifconfig wlp0s20f0u3 up
```
